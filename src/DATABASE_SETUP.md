## ⚡ Quick Start (3 Bước)

### Bước 1: Truy cập Supabase Dashboard
1. Mở trình duyệt và truy cập: **https://supabase.com/dashboard**
2. Đăng nhập vào tài khoản của bạn
3. Chọn project: `dqylfuxfsbgvtenypolu`

### Bước 2: Chạy SQL Script
1. Trong dashboard, click vào **SQL Editor** (menu bên trái)
2. Click nút **New Query** để tạo query mới
3. Mở file `/supabase-setup.sql` trong project này
4. **Copy toàn bộ nội dung** file SQL
5. **Paste** vào SQL Editor trong Supabase
6. Click nút **Run** (hoặc nhấn `Ctrl/Cmd + Enter`)
7. Đợi cho đến khi thấy thông báo "Success"

### Bước 3: Kiểm Tra & Xác Nhận
1. Vào **Table Editor** (menu bên trái)
2. Xác nhận các bảng sau đã được tạo:
   - ✅ `kv_store_c138835e` (Bảng chính - BẮT BUỘC)
   - ✅ `hostings`
   - ✅ `projects`
   - ✅ `passwords`
   - ✅ `categories`
   - ✅ `codex`
   - ✅ `logs`
   - ✅ `email_settings`

3. Quay lại ứng dụng và vào **Cài Đặt → Debug Panel**
4. Click **"Kiểm tra kết nối"**
5. Xác nhận thấy thông báo "✅ Kết nối thành công!"

## 🏗️ Cấu Trúc Database

### Bảng Chính: `kv_store_c138835e`

Đây là bảng **QUAN TRỌNG NHẤT** - lưu trữ tất cả dữ liệu dưới dạng key-value.

```sql
CREATE TABLE kv_store_c138835e (
  key TEXT PRIMARY KEY,
  value JSONB NOT NULL,
  created_at TIMESTAMPTZ DEFAULT NOW(),
  updated_at TIMESTAMPTZ DEFAULT NOW()
);
```

**Format dữ liệu:**
- `hosting:{id}` → Dữ liệu hosting
- `project:{id}` → Dữ liệu project
- `password:{id}` → Dữ liệu password
- `category:{id}` → Dữ liệu category
- `codex:{id}` → Dữ liệu code snippet
- `log:{id}` → Dữ liệu log

### Các Bảng Phụ (Optional)

Các bảng này được tạo để hỗ trợ query và báo cáo, nhưng **KHÔNG BẮT BUỘC** vì hệ thống sử dụng KV store làm chính:

1. **hostings** - Quản lý hosting/domain
2. **projects** - Quản lý dự án
3. **passwords** - Quản lý mật khẩu
4. **categories** - Danh mục password
5. **codex** - Code snippets
6. **logs** - Activity logs
7. **email_settings** - Cấu hình email

## 🐛 Khắc Phục Lỗi Thường Gặp

### Lỗi 1: "Health check failed: 404"

**Nguyên nhân:** Supabase Edge Function chưa được deploy hoặc chưa khởi động.

**Giải pháp:**
1. Hệ thống tự động deploy edge function khi bạn mở project
2. Đợi 10-30 giây để edge function khởi động
3. Refresh trang và thử lại
4. Nếu vẫn lỗi, kiểm tra Console logs

### Lỗi 2: "Không thể tải dữ liệu từ Supabase"

**Nguyên nhân:** Bảng `kv_store_c138835e` chưa được tạo.

**Giải pháp:**
1. Làm theo **Quick Start** ở trên
2. Chạy SQL script trong `/supabase-setup.sql`
3. Xác nhận bảng đã được tạo trong Table Editor
4. Refresh trang web

### Lỗi 3: "CORS error" hoặc "Network error"

**Nguyên nhân:** Trình duyệt block request hoặc network issue.

**Giải pháp:**
1. Kiểm tra kết nối internet
2. Tắt extensions trình duyệt (AdBlock, Privacy Badger, etc.)
3. Thử trình duyệt khác (Chrome, Firefox, Edge)
4. Xóa cache trình duyệt và refresh

### Lỗi 4: "Authentication failed"

**Nguyên nhân:** Anon key không hợp lệ.

**Giải pháp:**
1. File `/utils/supabase/info.tsx` là **AUTOGENERATED** - không được sửa
2. Nếu key hết hạn, hệ thống sẽ tự động tạo lại
3. Không cần làm gì cả

### Lỗi 5: Empty data (0 records cho tất cả modules)

**Nguyên nhân:** Database rỗng hoặc mới setup.

**Giải pháp:**
1. Đây là **BÌNH THƯỜNG** nếu bạn mới setup
2. Thử thêm một hosting hoặc project mới
3. Kiểm tra Debug Panel để xác nhận kết nối
4. Nếu thêm được dữ liệu → Hệ thống hoạt động tốt!

## 🔧 Advanced: Database Management

### Reset Database (Xóa tất cả dữ liệu)

⚠️ **CẢNH BÁO**: Thao tác này xóa TOÀN BỘ dữ liệu và không thể khôi phục!

```sql
-- Xóa tất cả dữ liệu trong KV store
DELETE FROM kv_store_c138835e;

-- Hoặc xóa luôn bảng (sau đó phải chạy lại setup script)
DROP TABLE IF EXISTS kv_store_c138835e CASCADE;
```

### Backup Database

**Cách 1: Export từ ứng dụng**
1. Vào **Cài Đặt**
2. Click **"Xuất Dữ Liệu"**
3. Lưu file JSON

**Cách 2: Export từ Supabase**
```sql
-- Xuất toàn bộ KV store
SELECT * FROM kv_store_c138835e;
```

### Restore Database

**Cách 1: Import vào ứng dụng**
1. Vào **Cài Đặt**
2. Click **"Nhập Dữ Liệu"**
3. Chọn file JSON đã backup

**Cách 2: Sử dụng SQL**
```sql
-- Insert data mẫu
INSERT INTO kv_store_c138835e (key, value) VALUES
('hosting:1', '{"id":"1","name":"Test Hosting","domain":"test.com",...}'),
('project:1', '{"id":"1","name":"Test Project","client":"ABC",...}');
```
